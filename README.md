# Calculator

## A clean, responsive calculator built with HTML, CSS, and vanilla JavaScript. Deployed and live at:

https://calculator.muhammadrizkikurniaputra.com/

## Features

- Basic arithmetic operations: Addition (+), Subtraction (−), Multiplication (×), Division (÷)
- Decimal number support
- Clear (Reset) and equals (=) functionality
- Responsive design — works on mobile, tablet, and desktop
- No external dependencies — 100% vanilla JavaScript
- Fast and lightweight
